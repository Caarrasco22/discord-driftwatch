# Compliance

Driftwatch follows defensive and policy-conscious design rules:

- Official Discord APIs only.
- No selfbots.
- No user tokens.
- No Message Content Intent.
- No Guild Presences Intent.
- Avoid Guild Members Intent.
- Respect Discord rate limits.
- Minimal permissions by default.
- No offensive functionality.

Operators are responsible for using Driftwatch only in servers where they are authorized to perform security auditing.
